# Assets Folder

Place your images and videos here for the README.

## Recommended Files

### Screenshots
- `screenshot-main.png` - Main SAGE interface
- `screenshot-voice.png` - Voice command in action
- `screenshot-task.png` - Task execution view
- `screenshot-thinking.png` - AI thinking section

### Demo Video/GIF
- `demo.gif` - Short GIF demo (recommended: 10-30 seconds)
- `demo.mp4` - Full demo video

## How to Add to README

### For Images:
```markdown
![Screenshot](assets/screenshot-main.png)
```

### For GIF Demo:
```markdown
![SAGE Demo](assets/demo.gif)
```

### For YouTube Video:
```markdown
[![SAGE Demo](https://img.youtube.com/vi/YOUR_VIDEO_ID/maxresdefault.jpg)](https://www.youtube.com/watch?v=YOUR_VIDEO_ID)
```

## Tips for Good Screenshots

1. **Clean desktop** - Hide personal files
2. **Good resolution** - At least 1280x720
3. **Show features** - Capture SAGE doing something interesting
4. **Consistent size** - Keep all screenshots similar dimensions

## Tips for Demo Video

1. **Keep it short** - 1-2 minutes max
2. **Show key features**:
   - Wake word activation
   - Voice command processing
   - Text input
   - App opening
   - Email/WhatsApp
   - Calculator
3. **Add captions** - Helps viewers understand
4. **Good audio** - Clear voice commands
